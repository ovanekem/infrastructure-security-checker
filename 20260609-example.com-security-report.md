## Vulnerability test results
- ✅ **BEAST attack**: PASS — TLSv1.0/CBC preconditions were not both present.. Remediation: Disable TLSv1.0 and CBC suites for modern clients.
- ✅ **BEAST attack (CVE-2011-3389)**: PASS — CVE-2011-3389: TLSv1.0/CBC preconditions not met — not vulnerable.. Remediation: Disable TLSv1.0 and all CBC cipher suites. Prefer TLSv1.2+ with ECDHE/GCM suites.
- ✅ **POODLE (SSLv3) (CVE-2014-3566)**: PASS — SSLv3/CBC exposure not detected.. Remediation: Disable SSLv3 entirely and reject downgrade attempts.
- ✅ **POODLE (TLS) (CVE-2014-8730)**: PASS — Server rejected CBC-only ClientHello — no CBC negotiation possible.. Remediation: No action required.
- ✅ **Downgrade attack prevention (CVE-2014-3566/TLS_FALLBACK_SCSV)**: PASS — Server rejected downgraded ClientHello with alert description 40.. Remediation: No action required.
- ✅ **SSL/TLS compression / CRIME (CVE-2012-4929)**: PASS — Compression is disabled (confirmed via -state output).. Remediation: No action required.
- ✅ **RC4 (CVE-2013-2566/CVE-2015-2808)**: PASS — No RC4 support detected in tested cipher set.. Remediation: Disable RC4 cipher suites.
- ✅ **Heartbleed (CVE-2014-0160)**: PASS — Server did not complete handshake — Heartbeat extension not supported or rejected.. Remediation: No action required.
- ✅ **Ticketbleed (CVE-2016-9244)**: PASS — Server did not complete handshake — session ticket extension not supported or rejected.. Remediation: No action required.
- ✅ **LUCKY13 (CVE-2013-0169)**: PASS — Server does not negotiate CBC cipher suites — LUCKY13 timing side-channel precondition not met.. Remediation: No action required.
- ✅ **LOGJAM (CVE-2015-4000)**: PASS — Server does not accept DHE or DHE_EXPORT cipher suites — LOGJAM precondition not met.. Remediation: No action required.
- ✅ **FREAK (CVE-2015-0204)**: PASS — Server rejected all RSA_EXPORT cipher suites — FREAK precondition not met.. Remediation: No action required.
- ✅ **SWEET32 (CVE-2016-2183 / CVE-2016-6329)**: PASS — Server rejected all 64-bit block cipher suites (3DES/DES/IDEA) — SWEET32 precondition not met.. Remediation: No action required.

- ✅ **Opossum (CVE-2025-49812)**: PASS — Server rejected the conflicting ClientHello with TLS alert (level=2, desc=40) — expected behaviour, not vulnerable.. Remediation: No action required.
